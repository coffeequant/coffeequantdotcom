You are driving from home to office, and you want to take the shortest path. There are two roads and bridges built between them. The numbers mentioned in the diagram is the time taken (assume minutes) for driving through that portion. Knowing all this information your program should generate the least time path for any set of input paths.




For example above path would be represented as Input: 49 11 31 4 89 21 41 3 25 10 8 0


“0” at the end is added for simplicity. The output should be

Output: [(Country,11),(Cross,31),(City,4),(Cross,21),(Country,3),(Country,8)]

*Main> init $ minimumPath [(RoadJoint 49 11 31),(RoadJoint 4 89 21),(RoadJoint 41 3 25),(RoadJoint 10 8 0)]

[(Country,11),(Cross,31),(City,4),(Cross,21),(Country,3),(Country,8)]

data Label = City | Country | Cross deriving (Show)

type QPath = [(Label,Int)]



data RoadJoint = RoadJoint Int Int Int deriving (Show)

type RoadSystem = [RoadJoint]



move::(QPath,QPath) -> RoadJoint -> (QPath,QPath)

move (pathCity,pathCountry) (RoadJoint a b c) =

let [priceCityPath,priceCountryPath] = map sum $ map (map snd) $ [pathCity,pathCountry]

updateCityPath = if ( (priceCityPath + a) <= (priceCountryPath + b + c)) then (City,a):pathCity else (Cross,c):(Country,b):pathCountry

updateCountryPath = if ( (priceCountryPath + b) <= (priceCityPath + a + c)) then (Country,b):pathCountry else (Cross,c):(City,a):pathCity

in (updateCityPath, updateCountryPath)



minimumPath :: RoadSystem -> QPath

minimumPath roadNetwork = let (bestCityPath, bestCountryPath) = foldl move ([],[]) roadNetwork

in if sum (map snd bestCityPath) <= sum (map snd bestCountryPath)

then reverse bestCityPath

else reverse bestCountryPath
