Problem Statement: Suppose you are traveling in an Indian Train that has 1 to N number of carts. Assume that you are locked up in nth cart where N > n. What is the expected value of N?

First Look: It's based on conditional probability. N is assumed random and the value “n” is given, from which you have to calculate expected value of N

P(N given n) = P(N intersection n) / P(n)

E[N] = summation from i = (n..N) ( i x P(N given n)).

The integral is simple to evaluate after we calc the conditional probability. For simplicity let's assume N = 100, and n = 5.

Let A = You are in Cart 5

B = There are 100 Carts

P(B|A) = P(A Intersection B) / P(A)

What is the probability of being dropped in 5th cart given that there are “i” carts? Answer = 96/100
