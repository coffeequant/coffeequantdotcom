If you still think Haskell is a shiny new kid on the block, you need to visit some links and convince yourself FP is here to stay. Apart from cool sounding names like Monads it seems extremely logical, and once you get it you realize how clever it is.

I read about automatic differentiation few months back, and with haskell as a tool it seemed a breeze to code it up. Here are some bits with comments

data ADDiff a =ADDiff a a deriving (Eq,Show)

sqr x = x * x

In short automatic differentiation adds an extra attribute to any variable. Any simple variable like x will have it's derivative attached to it. Building up the chain rules on top of it is a chain process.

In haskell all you need to do is change the basic functions like exp, sin, log and tell them to carry this extra property. This might sound daunting, but anyway skip it...

instance Floating x => Floating (ADDiff x) where
pi =ADDiff pi 0
exp (ADDiff x x') =ADDiff (exp x) (x' * exp x)
log (ADDiff x x') =ADDiff (log x) (x' / x)
sqrt (ADDiff x x') =ADDiff (sqrt x) (x' / (2 * sqrt x))
sin (ADDiff x x') =ADDiff (sin x) (x' * cos x)
cos (ADDiff x x') =ADDiff (cos x) (x' * (- sin x))
asin (ADDiff x x') =ADDiff (asin x) (x' / sqrt (1 - sqr x))
acos (ADDiff x x') =ADDiff (acos x) (x' / (- sqrt (1 - sqr x)))
atan (ADDiff x x') =ADDiff (atan x) (x' / (- sqrt (1 - sqr x)))

sinh (ADDiff x x') =ADDiff (sinh x) (x' / (- sqrt (1 - sqr x)))
cosh (ADDiff x x') =ADDiff (cosh x) (x' / (- sqrt (1 - sqr x)))
asinh (ADDiff x x') =ADDiff (asinh x) (x' / (- sqrt (1 - sqr x)))
atanh (ADDiff x x') =ADDiff (atanh x) (x' / (- sqrt (1 - sqr x)))
acosh (ADDiff x x') =ADDiff (acosh x) (x' / (- sqrt (1 - sqr x)))

instance Num a => Num (ADDiff a) where
ADDiff x x' +ADDiff y y' =ADDiff (x+y) (x'+y')
ADDiff x x' *ADDiff y y' =ADDiff (x*y) (y'*x + x'*y)
fromInteger x =ADDiff (fromInteger x)

negate (ADDiff x x') =ADDiff (negate x) (negate x')
signum (ADDiff x _ ) =ADDiff (signum x) 0
abs (ADDiff x x') =ADDiff (abs x) (x' * signum x)

instance Fractional x => Fractional (ADDiff x) where
fromRational x =ADDiff (fromRational x) 0
recip (ADDiff x x') =ADDiff (recip x) (- x' / sqr x)
