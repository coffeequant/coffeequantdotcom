Brownian Motion Puzzle

Consider a random walk which has two barriers “a” and “-b”. The random walk starts at 0 and can go either way. What is the probability that it hits “a” before it hits “b”. Here's a pretty crude image below



If I am standing on “a” what is the probability it will hit “a” before it hits “b”. Answer = 1

If I am standing on “b”, what is the probability it will hit “a” before it hits “b”

Answer = 0

Now these look like boundary conditions what do I apply here. I apply backward kolgomorov equation.







In our case drift is 0, so there are only two terms. What about the time derivative? That is also 0, coz there is no time derivative!

So it all comes down to simple second derivative which is easy to solve. If second derivative is 0, I can say that p = alpha x + beta, which satisfies second derivative = 0 condition.

Applying the boundary conditions here.

Alpha a + beta = 1 ….(1)

Alpha b + beta = 0 ….(2)

Solve for alpha and beta using ..(1) and ..(2)

Using simple solving we get alpha => (1-beta)b/a+ beta = 0

or b/a + beta (1-b/a) = 0 or beta = (b/a) / (b/a-1) or b/(b-a), alpha = (1-beta)/a

or alpha = (1-(b/b-a))/a = (b-a-b)/(b-a)/a = 1/(a-b)

p = (x-b)/a-b, (Answer)
